# 네모왕국 흑vs백 4

- 개발일시
> (2017 ~ 2017)
- 개발 엔진
> GameMaker Studio 1.4
- 개발 언어
> GameMakerLanguage
- 개발 인원
> 본인(1명)
- 배경 소개
> 고등학교 시절, 다시 네모왕국을 만들고 싶어져서 만들게 된 게임
>
- 게임 소개
> 싱글 기반의 게임.
>
> p2p 형식의 멀티플레이 배틀 구현
>
> 100개의 유닛을 만든 지금까지 네모왕국의 유닛들 대부분이 나온 게임
