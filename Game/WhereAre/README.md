# 으악! 어디야!

- 개발일시
> (2017 ~ 2017)
- 개발 엔진
> GameMaker Studio 1.4
- 개발 언어
> GameMakerLanguage
- 개발 인원
> 본인 외 고등학교 친구 2명(3명)
- 담당 역할
> 해당 게임에서 개발을 총괄하였음.
- 배경 소개
> 고등학교 시절, 고등학교에서 진행한 게임잼에 참여하여 개발한 게임.
> 대상을 수상하였음

- 게임 소개
> P2P통신 시스템의 멀티플레이 기반의 게임.
>
> 바다 한 복판에서 튜브를 낀 4명의 남자가 골인 지점까지 경주하는 게임
>
> 아이템 및 물리엔진이 반영되어 있음.
>
